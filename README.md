# ZenSafe

ZenSafe is an AI-driven digital well-being app for children. It encourages healthy smartphone usage with games, learning, and rewards.

## Features
- Track smartphone usage
- Reward system with stars and chocolates
- Fun learning games: Doodle Land, Word Scramble, Time Wizz
- Safety tools for healthy digital habits

## Tech Stack
- Flutter (Dart)
- Supabase backend
- Cross-platform: iOS, Android, Web, Windows, Linux
